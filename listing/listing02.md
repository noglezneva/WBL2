Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

При вызове функции test(), значение переменной x устанавливается равным 1, затем выполняется отложенная функция defer, которая инкрементирует значение переменной x на 1. Затем функция test() возвращает значение переменной x, которое равно 2.

При вызове функции anotherTest(), значение переменной x инициализируется как 0, затем выполняется отложенная функция defer, которая инкрементирует значение переменной x на 1. Затем функция anotherTest() возвращает исходное значение переменной x, которое равно 1.
```
