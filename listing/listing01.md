Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод: [77 78 79]
В программе создается массив a с пятью элементами типа int. Затем объявляется срез b, который является представлением части массива a с индексами от 1 до 3 (элементы [77, 78, 79]). Этот срез присваивается переменной b. Затем срез b выводится на экран с помощью функции fmt.Println.

```
